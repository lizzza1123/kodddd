# Генератор случайных паролей

**Автор:** [Батищева Елизавета]

Простое GUI-приложение на Python (Tkinter) для генерации случайных паролей с гибкими настройками и сохранением истории в JSON.

## Функциональность
- Выбор длины пароля через ползунок (1–128 символов).
- Чекбоксы для включения:
  - Букв (A-Z, a-z)
  - Цифр (0-9)
  - Специальных символов
- Гарантированное присутствие хотя бы одного символа из каждой выбранной группы.
- Отображение сгенерированного пароля.
- История последних паролей (до 50 отображается, все сохраняются в JSON).
- Кнопка очистки истории.
- Валидация: нельзя сгенерировать пароль, если не выбран ни один тип символов.
- Сохранение истории в файл `history.json`, автоматическая загрузка при запуске.

## Требования
- Python 3.6 или выше
- Стандартная библиотека Tkinter (обычно идёт вместе с Python)
